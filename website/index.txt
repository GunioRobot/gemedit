h1. Gem Editor

h1. &#x2192; 'gemedit'


h2. What

Gemedit lets you quickly open up the source for a gem in your favorite editor.

h2. Installing

<pre><code>sudo gem install gemedit</code></pre>

If you would like some basic command completion for installed gems add the following to your <code>~/.bashrc</code> or <code>~/.profile</code>:
<pre><code>complete -C "/usr/bin/gemedit --complete" gemedit</code></pre>

h2. Demonstration of usage

gemedit gemedit
